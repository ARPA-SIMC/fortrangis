New release 2.3
     dcesari - 2014-Feb-25 15:11   -   FortranGIS
This release supports managing of gdal metadata and RPC info. An interface to the readosm library for reading OpenStreetMap data has been introduced. Some bugs have been fixed and documentation has been improved.

Support for readosm in svn
     dcesari - 2013-Aug-09 17:57   -   FortranGIS
An experimental Fortran interface to the readosm library for reading OpenStreetMap vector data is in FortranGIS svn.

FortranGIS version 2.0 released
     dcesari - 2012-Feb-07 09:56   -   FortranGIS
A new version of FortranGIS is available, interface to Gdal (raster GIS dataset I/O) is now complete, including write access.

First package released
     dcesari - 2011-Jul-22 17:04   -   FortranGIS
The FortranGIS project has released its first source package. It builds on Linux and other UNIX-like systems. It includes also a spec file for building binary rpm packages.
